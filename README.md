# TradeTrack_V1

A trading backend application built with Java, Spring Boot, and MySQL.

## 🚀 Features
- Real-time trading data
- User authentication and management
- Order and trade tracking
- MySQL database integration

## ⚙️ Tech Stack
- Java 17 + Spring Boot
- MySQL
- REST APIs
- Git & GitHub for version control

## 🧠 How to Run
1. Clone the repository  
   `ash
   git clone https://github.com/khaledbakarman/TradeTrack_V1.git
   `
2. Configure your MySQL database in application.properties.
3. Run TradeTrackProBackendApplication.java to start the server.

## 💾 Version Control
- All changes are automatically tracked and pushed to GitHub.
- Each update is committed with a clear message.
